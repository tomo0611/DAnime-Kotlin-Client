# DAnime Links

## About
This is a collection of links for the dAnime Store. Created on 6/6/2025.

## Notice
The links are for the PC version of the dAnime Store.  
Remove `_pc` from the URL to access the mobile version.  
But be aware that some pages may not be compatible with mobile devices.
